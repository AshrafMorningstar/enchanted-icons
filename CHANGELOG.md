# Changelog

All notable changes to the "CodeCharm Icons" extension will be documented in this file.

## [1.0.0] - 2026-01-19

### Added

- Initial release of CodeCharm Icons
- 4 color variants: Base, Light, Soft, and Warm
- 100+ file type icons for popular languages and frameworks
- 20+ folder icons with open/closed states
- Customization settings for explorer arrows and folder styles
- Custom icon association support for languages, extensions, files, and folders
- Support for JavaScript, TypeScript, Python, Java, C++, Go, Rust, and more
- Web framework icons: React, Vue, Angular, Svelte
- Backend framework icons: Node.js, Express, Django, Flask
- Build tool icons: Webpack, Vite, Rollup, Gulp
- Testing framework icons: Jest, Mocha, Cypress, Playwright
- Configuration file icons: .env, .gitignore, eslint, prettier, tsconfig
- Documentation icons: Markdown, README, LICENSE
